Pong - Smash! _Ver.0.9

Press [enter/space] key to start game.

Move keys: [up][down] | [left][right]

[l-shift] key slows down paddle move speed.

[space] key activates smash ability.

Press [p] key while in gameplay mode to control paddle or make ai plays for you!

*score system is broken!
